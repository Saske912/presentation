# 🚀 Быстрый старт - Масштабирование диаграмм

## Управление масштабом диаграмм:

### 🖱️ Кнопки (появляются при наведении):
```
┌─────────────────────┐
│  [+]  [⟲]  [−] 100% │  ← В правом верхнем углу диаграммы
└─────────────────────┘

[+]  - Увеличить масштаб
[⟲]  - Сбросить в 100%
[−]  - Уменьшить масштаб
100% - Текущий масштаб
```

### ⌨️ Клавиатура + мышь:
- **Ctrl + колесико вверх** → Увеличить (zoom in)
- **Ctrl + колесико вниз** → Уменьшить (zoom out)
- **Mac**: Command вместо Ctrl

### 📜 Прокрутка:
- Обычное колесико (без Ctrl) → прокрутка диаграммы
- Shift + колесико → горизонтальная прокрутка

## 🎯 Типичные сценарии:

### Большая диаграмма не помещается:
1. Уменьшите масштаб (70-80%) чтобы увидеть всю диаграмму
2. Или прокрутите для просмотра разных частей

### Текст мелкий и плохо читается:
1. Увеличьте масштаб (150-200%)
2. Прокручивайте для просмотра разных частей

### Быстрый обзор → детали:
1. Сначала уменьшите для общей картины (50-70%)
2. Затем увеличьте для деталей (150-200%)
3. Сброс [⟲] для возврата к 100%

## 📊 Диапазон масштабирования:
- **Минимум**: 30% (0.3x)
- **По умолчанию**: 100% (1.0x)
- **Максимум**: 300% (3.0x)

## 🔥 Pro Tips:
- 💡 Ctrl+Scroll даёт более плавное управление
- 🎯 Каждая диаграмма сохраняет свой масштаб независимо
- 🌈 Кнопки появляются только при наведении - не мешают просмотру
- ⚡ Анимации делают масштабирование плавным и приятным
